Fundamentals of Data Structures

The objective of the course is to: Solve problems using data structures such as linear lists, stacks, queues, hash tables, binary trees, heaps, tournament trees, binary search trees, and graphs, and writing programs for these solutions. Solve problems using algorithm design methods such as the greedy method, divide and conquer, dynamic programming, backtracking, and branch and bound and writing programs for these solutions. Develop proficiency in the specification, representation, and implementation of Data Types and Data Structures.
